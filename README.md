\# Rikiavimo algoritmų lyginamoji analizė



Šioje saugykloje pateiktas C++ programos kodas, skirtas įterpimo rikiavimo ir suliejimo rikiavimo algoritmų palyginimui.



Programa testuoja algoritmus su 5000, 10000 ir 50000 elementų duomenų rinkiniais. Naudojami atsitiktiniai, atvirkščiai surikiuoti ir jau surikiuoti duomenys.



Duomenų failai atskirai nenaudojami, nes testavimo duomenys sugeneruojami pačioje programoje.



Failai:

\- main.cpp – programos kodas;

\- rezultatai.txt – eksperimento rezultatai.
